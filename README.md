# تبدیل گفتار به متن فارسی

این یک پروژه ساده برای تبدیل گفتار به متن فارسی است که قابلیت یادگیری از ویرایش‌های کاربر را دارد.

## اهداف پروژه:
- تبدیل گفتار به متن فارسی
- امکان ویرایش متن توسط کاربر
- ذخیره و استفاده از ویرایش‌ها برای بهبود سیستم

## ساختار:
- `frontend/`: کدهای سمت کاربر
- `backend/`: کدهای سمت سرور
- `data/`: داده‌های ویرایش‌شده
